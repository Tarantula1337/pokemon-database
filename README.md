# pokemon-database

## What this is about?

With this project I aim to learn some socket communication between server and client while also implementing something useful for myself.

## What's the goal?

The goal of this project is to build an application with which I can see what Pokemon I can catch, given the editions I own.

## Why FreePascal?

I learned it a bit in school and I always hear that you can build every program with every language, so I will use FreePascal to make it really hard to come back to this project in the future :3
